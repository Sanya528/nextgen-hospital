About the Platform

Next-Gen Hospital Appointment and Patient Data Platform is a smart healthcare solution designed to modernize hospital operations and improve patient experience.

The system allows patients to securely register, store their medical profile, and book appointments with doctors online through an intuitive web interface.

Administrators can manage doctors, monitor patient appointments, and maintain centralized hospital data efficiently.

Technologies Used

  • Flask – Backend Framework
  • HTML & CSS – Frontend UI
  • Python Lists/Dictionaries – Local Data Storage (Prototype Phase)
  • AWS Ready Architecture – EC2, DynamoDB, SNS (Future Deployment)

Project Objective

To build a secure, scalable, and cloud-ready hospital platform capable of managing patient data and doctor appointments with real-time updates and future AWS integration.
